## üîê Account Security Basics

### Password and PIN Security

#### **Create Strong Credentials**
**For Mobile App PIN (6 digits):**
- **Don't use obvious numbers** like 123456, 000000, or your birthdate
- **Avoid repeating numbers** like 111111 or 222222
- **Don't use easily guessable patterns** like 121212 or 456456
- **Choose something meaningful but not obvious** to others
- **Write it down somewhere safe** in case you forget

**For Web Dashboard Password:**
- **At least 8 characters** long
- **Mix of uppercase and lowercase letters** (AaBbCc)
- **Include numbers** (0-9)
- **Add special characters** (!@#$%^&*)
- **Avoid personal information** (name, birthdate, phone)
- **Don't reuse passwords** from other accounts

#### **Protect Your Login Information**
- **Never share your PIN or password** with anyone
- **Don't write passwords** where others can see them
- **Change compromised credentials** immediately
- **Use different passwords** for different systems
- **Log out completely** when finished working

### Device Security

#### **Mobile Device Protection**
- **Keep your device with you** or stored securely
- **Use device lock screen** (pattern, PIN, or fingerprint)
- **Enable automatic screen lock** after 1-2 minutes
- **Install security updates** when prompted
- **Report lost or stolen devices** immediately

#### **Physical Security**
- **Don't leave devices unattended** in public places
- **Store devices securely** when not in use
- **Be aware of your surroundings** when using devices
- **Use privacy screens** if working in public areas
- **Secure devices before transport**

## üåê Internet and Network Security

### Safe Internet Practices

#### **Public WiFi Safety**
- **Avoid public WiFi** when possible for work tasks
- **Use mobile data** instead of public hotspots
- **If using public WiFi** is necessary:
  - **Verify network names** to avoid fake hotspots
  - **Don't access sensitive information** on public networks
  - **Use VPN** if provided by your organization
  - **Log out completely** when finished

#### **Home Network Security**
- **Secure your WiFi** with strong password
- **Change default router passwords**
- **Keep router firmware updated**
- **Use WPA3 encryption** if available
- **Separate guest network** for visitors

#### **Recognizing Security Threats**
- **Phishing emails** asking for login information
- **Suspicious links** in messages or emails
- **Urgent requests** for account verification
- **Unexpected attachments** from unknown senders
- **Pop-up warnings** about security issues

### Data Protection

#### **Safe Data Handling**
- **Save work frequently** to prevent data loss
- **Back up important data** regularly
- **Don't store sensitive data** on personal devices
- **Use secure methods** for file sharing
- **Delete old data** no longer needed

#### **Data Transmission**
- **Check HTTPS** in web addresses for secure sites
- **Avoid public computers** for accessing SurveyLauncher
- **Clear browser data** on shared computers
- **Log out completely** from web sessions
- **Don't save passwords** in browsers

## üõ°Ô∏è Security Incident Response

### What to Do If Something Goes Wrong

#### **Lost or Stolen Devices**
**Immediate Actions:**
1. **Contact your supervisor** immediately
2. **Report to IT support** for account deactivation
3. **Change passwords** for all accessible accounts
4. **Document what happened** for your records
5. **Follow organization procedures** for equipment loss

**Prevention:**
- **Device tracking** enabled when possible
- **Regular backups** of important data
- **Secure storage** when not in use
- **Clear identification** on devices (contact info)

#### **Suspicious Activity**
**Warning Signs:**
- **Unusual login attempts** or account notifications
- **Unexpected password reset** emails
- **Strange messages** claiming to be from your organization
- **Account changes** you didn't make
- **Unknown devices** accessing your account

**Response Steps:**
1. **Change your password** immediately
2. **Review recent account activity**
3. **Report suspicious activity** to IT support
4. **Monitor your account** for additional issues
5. **Educate yourself** about the specific threat

#### **Phishing Attempts**
**How to Recognize Phishing:**
- **Urgent language** threatening account closure
- **Generic greetings** (Dear User instead of your name)
- **Poor grammar** or spelling mistakes
- **Unexpected attachments** or links
- **Requests for personal information**

**What to Do:**
- **Don't click links** in suspicious emails
- **Don't download attachments** from unknown senders
- **Verify requests** through official channels
- **Report phishing attempts** to IT security
- **Delete suspicious emails** without responding

## üö® Red Flags and Warning Signs

### Immediate Security Concerns

Contact your supervisor or IT support immediately if you notice:

#### **Account Security**
- **Unexpected password reset** emails
- **Login attempts from unusual locations**
- **Account changes you didn't make**
- **Messages about account suspension**
- **Security alerts** you don't recognize

#### **Device Security**
- **Slow device performance** without explanation
- **Unusual pop-ups or error messages**
- **Battery draining faster** than normal
- **Strange apps or programs** you didn't install
- **Device overheating** during normal use

#### **Data Security**
- **Files that won't open** or behave strangely
- **Missing or modified data** unexpectedly
- **Unusual error messages** during data entry
- **Slow app performance** with specific functions
- **Data that looks different** than expected

### Responding to Security Issues

#### **Immediate Steps**
1. **Stop using affected system** immediately
2. **Document what happened** with specific details
3. **Contact your supervisor** or IT support
4. **Change passwords** for potentially compromised accounts
5. **Preserve evidence** (don't delete files or messages)

#### **Follow-up Actions**
- **Monitor accounts** for continued unusual activity
- **Cooperate with security investigations**
- **Learn from the incident** to prevent recurrence
- **Share lessons learned** with team members
- **Update security practices** based on experience
